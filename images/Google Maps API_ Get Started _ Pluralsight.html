What you'll learn

This course introduces beginning to intermediate JavaScript developers to the Google Maps API, a powerful library that enables developers to harness the power of the Google Maps platform and build web mapping applications. The course shows how Google Maps works, how to embed a dynamic map within a web page, set its extent, add and symbolize geometry to display data, and take advantage of web services to add extra functionality to your applications.

About the author

Mark Lewin

Mark Lewin has been developing, teaching, and writing about software for over 16 years. His main interest is GIS and web mapping. Working for ESRI, the world's largest GIS company, he acted as a consultant, trainer and course author and a frequent speaker at industry events. He has subsequently expanded his knowledge to include a wide variety of open-source mapping technologies and a handful of relevant JavaScript frameworks including Node.js, Dojo and JQuery. Mark recently joined Oracles MySQL... more curriculum team to focus on creating great learning experiences for technical end users, but remains crazy about web mapping.
